# Getting Started

Mirotone React is a React component library for the Mirotone Design System. It provides a set of components and utilities to help you build your application for Miro.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Contributing](#contributing)
- [License](#license)
- [Versions](#versions)

## Installation

Install the package as a dependency on your own project by running:

```bash
npm install mirotone-react
```

```bash
yarn add mirotone-react
```

## Usage

All components and types are exported from the main package. You can import them as follows:

```tsx
import { Button, ButtonProps, Input, InputProps } from 'mirotone-react';
```
