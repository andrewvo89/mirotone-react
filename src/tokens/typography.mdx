## Font size

There are 6 font sizes available: small (12), medium (14), large (16), xlarge (20), xxlarge (24), xxxlarge (28). They are accissible via `tokens.typography.fontSize` the object.

## Font weight

There are 4 font weights available: normal (400), medium (500), bold (600), black (800). They are accissible via `tokens.typography.fontWeight` the object.

## Font

There are 2 fonts available:

- header ('Open Sans', -apple-system, BlinkMacSystemFont, Segoe UI, Helvetica, Arial, sans-serif,
  Apple Color Emoji, Segoe UI Emoji;),
- body ('Open Sans', -apple-system, BlinkMacSystemFont, Segoe UI, Helvetica, Arial, sans-serif, Apple Color Emoji,
  Segoe UI Emoji;)

They are accissible via `tokens.typography.font` the object.

## Usage

<DocBlockContainer>
  <Heading style={{ fontSize: tokens.typography.fontSize.medium }}>My custom heading</Heading>
  <Text style={{ fontWeight: tokens.typography.fontWeight.black }}>My custom text</Text>
</DocBlockContainer>

```tsx
import { AppCard, Centered, Heading, Spinner, tokens } from 'mirotone-react';

function App() {
  return (
    <>
      <Heading style={{ fontSize: tokens.typography.fontSize.medium }}>My custom heading</Heading>
      <Text style={{ fontWeight: tokens.typography.fontWeight.black }}>My custom text</Text>
    </>
  );
}
```
